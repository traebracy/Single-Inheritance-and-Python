# Single-Inheritance-w-Python-
A Python project I built to better understand object-oriented programming and how inheritance works between classes. This project focuses on single inheritance, where one class inherits attributes and behaviors from another class.

As I continue progressing in computer science, I’ve been putting a lot of focus into understanding not just how to write code, but how to structure it in a cleaner and more scalable way. Learning inheritance was a big step toward that.
